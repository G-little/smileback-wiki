#### 1.0 喜欢用户

##### 接口说明

##### 请求说明

| http 请求方式          |get             |
|:------------- |:---------------:|
| url      |/room_user/like |

#####  输入参数

| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| longtitude      | 是| double  |  经度 |  最多小数点后6位  |
| latitude      | 是| double  |  纬度 |   |
| roomId      | 是| string  |  空间ID |  |
| toUid      | 是| string  |  对方用户ID |  |

#####  错误说明
