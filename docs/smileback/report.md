#### 1.0 举报

##### 接口说明

添加留言信息

##### 请求说明

| http 请求方式          |post             |
|:------------- |:---------------:|
| url      |/report/add |

#####  输入参数

| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| reportType      | 是| string    | 举报类型|   |
| content      | 是| string    | 举报内容|   |
| picUrls      | 否| string[]    | 图片|   |
| toUid      | 否| int    | 被举报人|   |
| extType      | 否| byte    | 扩展参数类型|  1 封面  |
| extId      | 否| int    | 扩展参数ID|  extType  根据类型决定实际ID意义  |
